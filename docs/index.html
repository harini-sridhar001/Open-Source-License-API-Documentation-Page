Introduction

OSLI simplifies open-source license management by combining deterministic SPDX license data with AI-powered reasoning. It provides actionable insights, contextual risk assessments, and automated recommendations for developers, managers, and legal teams.

🔍 Discovery & Research

POST /v1/search — Smart Library Search

Discover libraries that meet your licensing requirements. Enter a natural language query and OSLI will return libraries compatible with your project’s policies.

ParameterTypeDescription
querystringNatural language description of what library you need
import requests
data = {"query": "Chart library for closed source"}
resp = requests.post("https://open-source-license-api.onrender.com/v1/search", json=data)
print(resp.json())

Find permissive alternatives to restrictive libraries (e.g., MIT instead of GPL), reducing legal risk while maintaining functionality.

💡 Tutorial: Easy Integration

import requests

def audit_licenses(deps):
    url = "https://open-source-license-api.onrender.com/v1/audit"
    payload = {"dependencies": deps}
    response = requests.post(url, json=payload)

    if response.json().get("risk_level") == "High":
        print("🚨 BUILD FAILED: Risky licenses detected!")
    else:
        print("✅ Licenses cleared.")

audit_licenses(["react", "ffmpeg", "lodash"])

⚠️ Error Handling & Correctness

  • 200 OK: Request successful
  • 400 Bad Request: Missing parameters or invalid JSON
  • 404 Not Found: Package not found in registry
  • 500 Internal Server Error: AI or registry issue
